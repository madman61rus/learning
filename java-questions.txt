2. Из каких символов может состоять имя переменной ( идентификатор )

   Не должен совпадать с ключевыми словами, или литералами. Не должен начинаться
   с цифры. Может содержать строчные и прописные буквы анг. алфавита, $, _ )

    Классы: существительное с заглавной буквы ( DogTest)
    Интерфейсы: прилагательное (..able),существительное с заглавной буквы
      CamelCase или I+CamelCase
    Методы: глагол/глагол+сущ+прилаг  camelCase
    Переменные: 1 слово/ camelCase / от. буквы
    Константы: все заглавные буквы, подчеркивание
    Пакеты: строчные буквы и подчеркивание
    Комментарии: //... , /*...*/ , /** ...*/

4. На какие основные группы можно разделить типы данных в Java

    На примитивные (содержит значение) и Ссылочные(содержит ссылку)

    Примитивные: byte, short, int, long,float,double,char,boolean
    Ссылочные: Аннотации, Массивы,Классы,Enumeration,Interface
